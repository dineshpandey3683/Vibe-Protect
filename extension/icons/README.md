# Icons

Place PNG icons here sized 16×16, 32×32, 48×48, and 128×128 as:

    icon16.png
    icon32.png
    icon48.png
    icon128.png

A quick-and-dirty placeholder: a solid `#FACC15` square with a white shield
glyph. Chrome will fall back to a default icon if these are missing, so the
extension still loads & works without them.
